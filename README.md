# Spotify Web Clone 🎵

A responsive Spotify Web Player UI clone built using HTML and CSS. This project replicates the core layout and design of Spotify's web interface including sidebar navigation, music player controls, playlists, and responsive card sections.

## Features
- Spotify inspired sidebar navigation
- Sticky top navigation bar
- Music player control section
- Playlist and chart cards UI
- Hover effects and interactive cursor behavior
- Responsive layout using Flexbox
- Clean and modern UI design

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Font Awesome Icons
- Google Fonts (Montserrat)

## Learning Objectives
This project was built to improve frontend development skills and understanding of:
- Layout structuring
- Flexbox positioning
- UI component design
- Responsive spacing
- Professional UI practices
